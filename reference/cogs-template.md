# COGS Reference — Formulas, Examples & Yield Factors

## Core formulas

- **Ingredient cost per serve** = (quantity used ÷ pack size) × pack price
- **Total COGS** = sum of all ingredient costs + packaging
- **Food cost %** = total COGS ÷ selling price × 100
- **Gross profit** = selling price − total COGS
- **GP%** = gross profit ÷ selling price × 100
- **Yield-adjusted cost** = raw cost ÷ yield %

|---|---|---|---|---|

## Yield factors — common ingredients

Apply when an operator provides raw purchase cost for a high-waste ingredient. Adjusted cost = raw cost ÷ yield %.

| Ingredient | Typical yield |
|---|---|
| Avocado | 75–80% |
| Whole fish (filleting) | 45–55% |
| Beef (trimming) | 70–80% |
| Whole chicken | 65–70% |
| Leafy greens | 85–90% |
| Onion (peeled) | 85% |

## Common costing errors to flag

- **Packaging omitted** — cup, lid, sleeve, takeaway bag can add $0.30–$0.50 per drink
- **Retail pricing used instead of invoice pricing** — will overstate food cost %
- **Yield not applied** — understates cost for high-waste ingredients
- **Stale invoice data** — if a key input has changed more than 10%, recosting is required
